# GuessMaster

GuessMaster is a simple desktop application built with Tkinter, allowing users to play a guessing game.

## Getting Started

### Prerequisites
- Python 3
- Tkinter (comes pre-installed with most Python installations)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/mudeitsi/PRODIGY_SD_02.git
   cd GuessMaster
   ```

2. Run the Tkinter app:
   ```bash
   python guess_master_app.py
   ```


3. Follow the instructions on the Tkinter window to input your guesses and enjoy the game!

## Features
- Simple graphical user interface (GUI) built with Tkinter.
- Users can input guesses and receive immediate feedback.
- Randomly generated numbers for each game.

## Usage
- Input your guess in the provided entry field.
- Click the "Submit Guess" button to see feedback on your guess.
- The game continues until you correctly guess the number.
